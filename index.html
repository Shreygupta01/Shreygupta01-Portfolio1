Making a TO-DO LIST

Here's a summary of a simple to-do list created using HTML and CSS: - The HTML file (`index.html`) includes a title, a task input field, an "Add" button, and a list to display tasks. - The CSS file (`style.css`) provides basic styling for the to-do list, including task items and buttons. - The JavaScript file (`app.js`) adds functionality to add tasks to the list and delete them. - Users can enter tasks, click the "Add" button, or press Enter to add tasks, and delete tasks by clicking the "Delete" button. - This results in a minimal and functional to-do list.
